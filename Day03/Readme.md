## 🛡️ Antivirus Basics

### 🔧 Main Functions:
- **Quarantine Malware:** Keeps threats away from your system.

### 🧪 Types of Analysis:
- **Static Analysis:** Examines files without running them.
- **Dynamic Analysis (Sandbox):** Runs files in a safe test environment.
